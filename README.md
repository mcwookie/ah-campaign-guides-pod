# ah-campaign-guides-pod
Arkham Horror campaign guides suitable for printing at lulu.com

This was put together by [u/fiendmish](https://www.reddit.com/user/fiendmish/) in this [post on Reddit](https://www.reddit.com/r/arkhamhorrorlcg/comments/rmxuf3/printed_campaign_guides/).  The files were stored in [Dropbox](https://www.dropbox.com/sh/4klj5so986ha341/AAA2abnGABbkdlM5neSYAg5Sa?dl=0).  I just put them on github so they wouldn't be lost.

From his "Lulu print settings used.rtf" file:

```
Lulu Settings used

Print Book: 
- Small Square (7.5 x 7.5 in / 190 x 190 mm), 
- Standard Color, 
- 80# White, 
- Coil Bound, 
- Glossy Cover
```
